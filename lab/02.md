# Bygg din egen webbserver

Målet med övningen är att bättre förstå hur ASP.NET ramverket fungerar genom att själv bygga en enkel webbserver.

**Övning:** Öppna Starter-projektet och studera koden.

Starta och anropa servern med hjälp av en webbläsare. Surfa till http://127.0.0.1:34567/ och kolla att du ser "Hello World".

Prova att anropa din server med Telnet som du gjorde tidigare.

### Verb och Path i Request

**Övning:** Ändra koden så att när man anropar http://127.0.0.1:34567/home så ska texten "Du är hemma" visas istället.

Notera var pathen från webbläsaren dyker upp.
 
```
GET /home HTTP/1.1
```
 
**Verifiera:** Prova att surfa till http://127.0.0.1:34567/home och du ska se texten. Surfa till någon annan path och du ska inte se texten.

**Övning:** Lägg till så att när man surfar till http://127.0.0.1:34567/sayhello så skrivs en hemsida med texten "Hej!" ut.

Notera verb och path ser ut vid anropet

```
GET /sayhello HTTP/1.1
```

**Verifiera:** Prova att surfa till http://127.0.0.1:34567/sayhello och du ska se texten "Hej!", ändrar du till /home ska du se den andra texten.

**Övning:** Formulär kan skicka in data till en webbserver. Lägg till följande kod efter att webbläsaren skriver ut "Hej!".

```
writer.WriteLine($"<form><label>Namn: <input name='name'></label><button>Skicka</button></form>");
```

När du laddar om sidan ska det dyka upp ett litet formulär med en knapp där du kan skriva ditt namn och trycka på Skicka-knappen.

Kolla vad som händer med pathen när du skriver ditt namn i rutan och trycker på skicka. Sidan laddas om och namnet från formuläret läggs på som en parameter.

Du borde se följande i adressfältet: http://127.0.0.1:34567/sayhello?name=Fredrik

Ändra koden så att den tar ut värdet efter "name=" och skriver ut "Hej Fredrik!". 

**Verifiera:** Om du skriver in ett annat namn i formuläret eller byter namn i URL:er ska det namnet visas. Så surfar du  till http://127.0.0.1:34567/sayhello?name=Ellen ska sidan säga "Hej Ellen!".
